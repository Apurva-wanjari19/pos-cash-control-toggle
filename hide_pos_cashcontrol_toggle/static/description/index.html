💡 Hide POS Cash Control

Give users the option to hide the Opening Cash Control popup in the Odoo POS system. Designed for flexibility and cleaner session starts.

🚀 Key Features

  • Adds a checkbox in POS settings: "Hide Opening Cash Control"
  • Automatically hides the popup when enabled
  • Toggle any time without restarting the POS app
  • Non-intrusive – does not modify any core files
  • Improves speed and flow of POS session startups

🧑‍💼 Usage Guide

  1. Navigate to Point of Sale → Settings
  2. Enable the "Hide Opening Cash Control" checkbox
  3. Save your changes
  4. Start a new POS session
  5. The Opening Cash Control popup will be skipped
  6. To show the popup again, uncheck the box and save
  7. This can be toggled anytime based on business needs
